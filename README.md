# Secure File Locker

## 🛠️ Features

✓ Password-based encryption.

✓ PBKDF2 key derivation.

✓ Random salt generation.

✓ Fernet authenticated encryption.

✓ Supports text, PDF, image, binary, and other file extensions.

✓ Cross-platform Python implementation.


## 🎯 Requirements

✅ Run `pip install cryptography` in the terminal to install the Cryptography library before running the program.

✅ Store the `main.py` and `encrypt_decrypt.py` files in the same folder.


## 🚀 Run

python main.py
